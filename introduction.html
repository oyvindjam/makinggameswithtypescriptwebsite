Introduction

We are going to create a 2D game from scratch using TypeScript.

This is not an introduction to programming, the book assumes prior programming experience and some knowledge about the Node.js ecosystem.

This is also not a step-by-step guide to making games, but rather a presentation of several topics and concepts, followed by actual working implementations.
