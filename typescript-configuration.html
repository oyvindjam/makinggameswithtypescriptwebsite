TypeScript configuration

The TypeScript compiler can be configured with a tsconfig.json file.

tsconfig.json
{
  "compilerOptions": {
    "strict": true,
    "noUnusedLocals": true,
    "noUnusedParameters": true,
    "noImplicitReturns": true,
    "noFallthroughCasesInSwitch": true,
    "sourceMap": true,
    "esModuleInterop": true
  }
}

We enable more compiler warnings and source-maps to make development and debugging easier.
